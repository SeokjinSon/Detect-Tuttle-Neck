# Readme

## **설명**
컴퓨터를 이용할 때 거북목인지 모른 채 계속해서 컴퓨터를 보고 있는 경우가 많이 있습니다. 그래서 현재 나의 목 상태가 거북목일 경우 거북목이라고 모니터에 표시가 된다면 인지할 수 있지 않을까라는 생각에 Google Teachable Machine을 사용해 간단히 만들어보았습니다.

## **사용 도구**
+ Google Teachable Machine
+ Python 3.7.3
+ opencv 4.5.1
+ tensorflow 2.4.1
